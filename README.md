# Calcul

Calcul is a simple calculator programming language, but with a twist.  
It can use text, functions can be created, etc.  
However, it can use text, but only in simple cases like `a<h,e,l,l,o> _O`(prints 'hello').  
while it may look scary, it's fairly simple actually:
* `a<>` --> add function;
* `h,e,l,l,o` --> the text to add together(yes you need to add each letter together, with . seperating);
* `_O` --> tells the interpreter that this line should be outputed.
